Sample Work

Free energy mini generator

This project demonstrates electromagnetic induction by using rotating magnets to generate electricity. A central arm with magnets moves between two fixed ring magnets, inducing current in a generator, which powers LEDs. It showcases the principles of magnetic energy conversion in a simple and efficient design.

Arduino LED Chaser

This project uses an Arduino to create a dynamic LED running light effect with customizable patterns. By programming precise timing sequences, it demonstrates microcontroller-based automation, making it ideal for decorative lighting and visual effects.
